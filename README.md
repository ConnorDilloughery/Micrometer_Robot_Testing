# Micrometer_Robot_Testing
This project is to test the accuracy and precision for the robots at SPT Microtechnologies. We want to show that the robot is performing repeatable cycles. 

This robot test used MetraLight's Rx Laser Micrometer. For more information, use the link below:
[Rx Laser Micrometer Manual](https://www.metralight.com/products/doc/rx/rx_userC.pdf)
